## 03 // Local Development (Instructions)
To build or update the site locally before deployment, follow these system protocols:

1. **Prerequisites:** Ensure you have Ruby, Bundler, and Jekyll installed on your machine.
2. **Install Dependencies:**
   ```bash
   bundle install
   ```
3. **Serve Locally:**
   ```bash
   bundle exec jekyll serve
   ```
